# Dr. Robert L. Woods MD - Concierge Psychiatry Website

A modern, professional website for Dr. Robert L. Woods MD, featuring concierge psychiatry services with personalized care delivered in the comfort of your own space.

## Features

- **Responsive Design**: Optimized for all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Contact Integration**: Direct phone and email links
- **Service Showcase**: Comprehensive psychiatric services
- **Patient Testimonials**: Rotating testimonials with auto-play
- **SEO Optimized**: Meta tags and structured data

## Tech Stack

- **Next.js 14**: React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first styling
- **Lucide React**: Beautiful icons
- **Google Fonts**: Inter & Merriweather typography

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```
